# ミニ読書ノートシステム

ブラウザだけで動く読書ノートです。データは **localStorage** に保存されます。

## ファイル構成

| ファイル | 役割 |
|---------|------|
| `index.html` | 画面の骨組み |
| `css/app.css` | レイアウト・見た目 |
| `js/store.js` | 読み込み・保存（ストレージ層） |
| `js/main.js` | フォーム・一覧・検索（UI層） |
| `list.html` | 登録一覧（テーブル表示） |
| `js/list.js` | 登録一覧ロジック |

## ブラウザで開く（おすすめ）

**Mac:** `ブラウザで開く.command` をダブルクリックすると、サーバーが立ち上がり **自動でブラウザ** が開きます。

**手動:** ターミナルで `library` に移動してから:

```bash
python3 -m http.server 8765
```

ブラウザのアドレス欄に **`http://127.0.0.1:8765/`** と入力して開いてください。

（複数ファイル構成は `http://` 経由の方が確実です。`index.html` 単体のダブルクリックでも動く環境があります。）
2. タイトル・著者・読了状態を入力して「追加」
3. 検索欄でタイトル・著者を絞り込み
4. 「削除」で1件ずつ消去

## 注意

別のブラウザ・端末とはデータは共有されません。
